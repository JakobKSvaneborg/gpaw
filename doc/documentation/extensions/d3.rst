=============
D3 correction
=============

Setup
-----

If GPAW is setup on Niflheim with the ``GPAW_venv.py`` script, 
then it must be run with the ``--dftd3`` flag to work like this::

    $ python3 gpaw_venv.py --dftd3 <venv-name>


Usage
-----

The D3 extension has one mandatory keyword, ``xc`` which is that exchange-correlation
functional to use.

An example of a system where the D3 correction adds meaningful correction 
is in the case of 2D bilayer systems.

    D3 requires the system to be either not periodic or periodic in all three directions. 
    If the system is only periodic in one or two directions a warning will appear and it will 
    assume that system is periodic in all three directions.


Output
------

The energy contribution from the D3 correction can by found in the output 
file just after the list of iterations. For the above example the relevant part of 
the output file will look as following::
    
    Energy contributions relative to reference atoms: (reference = -702828.048343)
    
    kinetic:          71.457107
    coulomb:         -50.449930
    zero:             -0.629602
    external:          0.000000
    xc:              -44.310857
    entropy:          -0.001037
    spinorbit:         0.000000
    --------extensions:---------
    D3 (xc=PBE):      -1.446148
    ----------------------------
    Free energy:     -25.380467
    Extrapolated:    -25.379948
